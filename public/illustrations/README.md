# 挿絵画像の配置先

各物語の挿絵画像をこのディレクトリ配下に配置します。

## 配置規約

```
public/illustrations/
├── {storyId}/
│   ├── cover.webp           # 表紙 (1024×1024 推奨)
│   ├── {scene-1}.webp       # 各シーン (1600×900 推奨)
│   ├── {scene-2}.webp
│   └── ...
```

`{storyId}` は `src/data/stories.ts` の `Story.id`、`{scene}` は `Page.scene` 値と完全一致させてください。

## 推奨スペック

- **フォーマット**: WebP (品質 80〜85)
- **解像度**: 表紙 1024×1024 (正方形) / シーン 1600×900 (16:9)
- **ファイルサイズ**: 1 枚 200KB 以下 (LCP NFR 2.5s 達成のため)
- **スタイル**: 子供 (3〜5 歳) 向けのやさしい表現
- **権利**: 本人作成 / CC0 / MIT 同等のもののみ。詳細は `LICENSE-illustrations.md` (Operations Flow で整備)

## 不在時の挙動

未配置のシーン (および表紙) は `placeholderEmoji` + `bg` 色面で自動フォールバック表示されます。
Delivery 期間中に順次配置していけば、配置済みシーンから順に実画像へ差し替わります。

## 想定総数

表紙 6 枚 + シーン 36 枚 = **約 42 枚**。
