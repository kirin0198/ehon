## 1. 方針

`public/illustrations/` 配下に配置する挿絵画像は、**リポジトリ本体の MIT
License（`LICENSE`）の対象外**として個別に権利情報を管理する。これは:

- 著作権者・ライセンス・出典を画像ごとに正確に追跡するため
- 第三者がリポジトリを fork した際に、コードと挿絵を別ライセンスで扱える
  ようにするため
- 子供向けプロダクトとして、原作 (パブリックドメイン) と挿絵 (新規制作物)
  の権利関係を明確にするため

### 受け入れる挿絵の条件

挿絵は以下のいずれかを満たすもののみリポジトリにコミットしてよい。

1. **オリジナル制作** — リポジトリの貢献者本人が描き下ろした作品で、
   MIT または CC0 / CC-BY 4.0 等のオープンライセンスで提供されるもの
2. **パブリックドメイン素材** — 著作権切れ、または明示的に PD 宣言された素材
3. **CC-BY / CC-BY-SA など適切なクレジット表記が可能なもの** — 表記を
   下表「画像ごとの権利情報」に確実に記録できること
4. **権利者から本リポジトリでの使用許諾を得たもの** — 許諾内容を文書化し、
   表に記載すること

> **禁止**: 出典不明 / SNS 拾い物 / 商用ストック画像の権利確認なし利用 /
> AI 生成画像で原画権利者の同意がないもの

## 2. 画像ごとの権利情報

挿絵を追加するときに本表へエントリを追加すること。**追加されていない挿絵
はコミット禁止**。

| パス             | 物語 | シーン | 制作者 / 出典 | ライセンス | クレジット表記 (必要時) | 備考                                               |
| ---------------- | ---- | ------ | ------------- | ---------- | ----------------------- | -------------------------------------------------- |
| _(エントリなし)_ | —    | —      | —             | —          | —                       | 表紙・本文挿絵は段階配置中。配置時に行を追加する。 |

### エントリ記入例

```
| public/illustrations/akazukin/cover.webp | 赤ずきん | cover | 山田太郎 (本リポジトリ貢献者) | CC-BY 4.0 | © 2026 山田太郎 / CC-BY 4.0 | 1024×640 webp |
| public/illustrations/momotaro/forest.webp | 桃太郎 | forest | 葛飾北斎『北斎漫画』 | Public Domain (CC0) | — | 1872 年版より引用 |
```

## 3. クレジット表記の表示場所

CC-BY 系ライセンスの場合、エンドユーザーが容易に確認できる場所に表記する。
本プロジェクトでは以下のいずれかで表記を行う:

- `README.md` の「Credits」セクション
- アプリ内フッターの「クレジット」リンク先
- 該当物語ビュアー画面の `<figure><figcaption>` 相当箇所 (将来検討)

## 4. 画像の推奨スペック

`README.md` の「挿絵画像の追加方法」を参照。要約:

- フォーマット: `.webp` (推奨) / `.png` (フォールバック)
- 表紙 (`cover.webp`): 横長 1024×640 程度、ファイルサイズ ≤ 200 KB 目安
- シーン挿絵: 横長 800×500 程度、ファイルサイズ ≤ 150 KB 目安
- 配色: 各物語の `coverColor` / `coverAccent` と調和

## 5. 違反発覚時の対応

権利侵害が疑われる挿絵が混入していると判明した場合:

1. 直ちに該当ファイルをリポジトリから削除する (`git rm` + 履歴再書き込みは
   状況に応じて検討)
2. 本ドキュメントの該当行を「削除済み」に更新
3. CHANGELOG.md に経緯を記録
4. 必要に応じて GitHub Issue を立て、第三者からの通報窓口とする
